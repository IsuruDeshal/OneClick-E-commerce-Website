Available Tags

How to Use Tags

  1. When adding/editing a product, select one or more tags from the dropdown
  2. Products will automatically appear in homepage sections based on selected tags
  3. You can assign multiple tags to show a product in multiple sections
  4. Example: Tag a laptop with both "shop-laptops" and "top-selling" to show it in both sections
